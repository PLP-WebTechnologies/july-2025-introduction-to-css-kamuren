Understanding the CSS Box Model

Elements in a web page are in boxes. This will show how margins, padding, and borders work to create space and define the layout.

Content Area

This is the actual content of the element. The text, images, and other elements live here. Everything else builds around this.

Padding Space

Padding creates space between the content and the border. Notice how the background color extends into the padding area.

Border & Margin

The border wraps around the content and padding. Margin creates space outside the border, separating elements from each other.
